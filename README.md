# 🏋️‍♂️ 健身動作查詢系統 (FitDex)

本專案為一款使用 C# Windows Forms 開發的本地端應用程式，旨在解決健身新手常見的「動作選擇困難」、「解剖部位不熟悉」以及「動作名稱不熟悉」等痛點。透過直覺的圖形化介面，使用者可以輕鬆查詢適合自己的健身動作，並建立專屬的個人訓練課表。

---

## ✨ 核心功能

* **會員與權限管理**：支援帳號註冊與密碼加密登入，並提供免登入的「訪客模式」（訪客點擊會員功能會自動引導登入）。
* **直覺化動作查詢**：點擊人體解剖圖上對應的肌肉部位（如 Chest, Back, Abs），即可快速篩選出該部位的訓練動作。
* **資料分頁與排序**：動作清單支援自訂每頁筆數、快速翻頁，且資料表可依各欄位動態排序。
* **視覺化圖文呈現**：點選動作即同步顯示「起始」與「結束」示範照片，並搭配圓餅圖統計目前篩選清單的訓練強度比例。
* **個人訓練課表**：會員專屬功能，可將動作加入「常用（★）」，並自由新增、刪除、上移、下移來排配專屬的今日課表。

## 🛠️ 使用技術

* **開發語言**：C#
* **介面框架**：Windows Forms (WinForms) / .NET
* **資料庫**：Microsoft SQL Server (LocalDB / `.mdf`)

## 🚀 如何執行

1. 將本專案 Clone 到本地電腦：
   git clone https://github.com/你的帳號/你的專案名稱.git
2. 使用 Visual Studio 開啟 `WinFrmApp_Workout.slnx` 方案檔。
3. 確認連線字串：開啟 `App.config` 或連線設定檔，確認 Connection String 正確指向專案資料夾中的 `.mdf` 資料庫檔案。
4. 按下 `F5` 或點擊「開始」編譯並執行專案。
